# 🛡️ 턴제 RPG 텍스트 게임 (Project: RpgGame)

## 🎮 게임 개요
용사가 되어 총 5개의 월드를 탐험하며, 각 월드의 보스를 무찌르고 황금 무기 조각을 모으세요.

모든 시련을 이겨내고 최종 보스로부터 세상을 구하는 텍스트 기반 턴제 RPG 게임입니다.

---

## ✨ 주요 특징
* **턴제 전투:** 용사와 몬스터가 한 턴씩 주고받는 고전적인 턴제 전투 시스템.
* **성장 시스템:** 레벨업을 통한 자유로운 스탯 분배(체력, 마나, 공격력, 마력)
* **전직 시스템:** '검사'에서 시작해 '성검사' 또는 '용기사'에 이르는 총 4단계의 직업 성장.
* **스테이지:** 총 5개의 월드, 월드당 10개의 스테이지 구성 (일반, 중간 보스, 월드 보스)
* **아이템:** 무기, 방어구, 반지 등 다양한 장비를 통한 추가 성장.
* **스킬:** 직업별 고유 스킬, 패시브, 궁극기를 활용한 전략적 전투.

---

## 🧑‍🤝‍🧑 직업 (전직) 시스템
플레이어는 '검사'로 시작하여 2개의 상위 계열로 성장할 수 있습니다.

* **기본 (Lv 1): 검사**
    * 밸런스형 초반 근접 전사. (공격 스킬 1, 방어 스킬 1)
* **1차 (Lv 10): 기사**
    * 방어/생존력 강화. 도발, 회복 스킬 보유. (공격 1, 방어 1, 패시브 1)
* **2차 (Lv 20) - 계열 분리**
    * **[방어/복합] 성기사:** 물/마 복합형. 회복, 신성 공격. (궁극기 추가)
    * **[공격/물리] 대검사:** 순수 물리 딜러. 한 방이 강하지만 MP 낮음. (궁극기 추가)
* **3차 (Lv 30) - 최종**
    * **[방어/마법] 성검사** (성기사 계열): 반사 피해, 높은 내구도, 장기전 특화. (궁극기 강화)
    * **[공격/필살] 용기사** (대검사 계열): 폭발적 공격력, '용의 분노' 스택. (궁극기 강화)

## 🗺️ 게임 시스템 상세

<details>
<summary><strong>🌍 월드 및 스테이지 구성</strong></summary>

* **게임 구조:** 총 5월드, 월드당 10 스테이지.
* `1~4, 6~9 스테이지`: 일반 몬스터 (1마리)
* `5 스테이지`: 중간 보스 몬스터
* `10 스테이지`: 월드 보스 몬스터 (클리어 시 황금 무기 조각 획득)
* `5월드 11 스테이지`: **최종 보스 몬스터**
</details>

<details>
<summary><strong>📈 레벨업 및 스탯</strong></summary>

* **전직 레벨:** 1 (검사) → 11 (기사) → 21 (성기사/대검사) → 31 (성검사/용기사)
* **스탯 성장:** 레벨업 시 **10 스탯 포인트** 부여. (1포인트당 능력치 10 증가)
* **분배 가능 스탯:** 체력, 방어력, 공격력, 마력
* **전직 보너스 (차등 부여):**
    * `검사 → 기사`: 공격력, 방어력, 체력
    * `기사 → 성기사`: 마력, 방어력, 체력
    * `기사 → 대검사`: 공격력, 체력, 방어력
    * `성기사 → 성검사`: 마력, 방어력, 체력
    * `대검사 → 용기사`: 공격력, 체력, 방어력
</details>

<details>
<summary><strong>⚔️ 기타 시스템 (몬스터, 아이템, 상태이상)</strong></summary>

* **몬스터 정보:**
    * 모든 몬스터는 **아스키 아트(ASCII Art)**로 표현됩니다.
    * 기본, 중간 보스, 보스, 최종 보스로 등급이 나뉩니다.
* **아이템 / 장비:**
    * `무기`
    * `방어구`
    * `반지`
* **상태이상 (n% 확률):**
    * `기절`
    * `감전`
</details>

## 👤 캐릭터 기능 목록
### 1. 캐릭터 기본 속성
초기 기본 스탯 :

- 최대 HP : 100

- 최대 MP : 50

- 물리 공격력 : 10

- 주문력 : 10

- 방어력 : 0 (기본값. 장비나 스탯 분배로 상승)

- 마법 저항력 : 0 (기본값. 장비나 스탯 분배로 상승)

상태 :

- 현재 HP, 현재 MP: 전투 및 스킬 사용에 따라 실시간으로 변동됩니다.

- 상태 이상 : 몬스터 스킬에 의해 '독', '기절', '침묵' 등의 상태에 걸릴 수 있습니다.

### 2. 캐릭터 행동
기본 공격 :

- MP 소모 없이 attackDamage 수치만큼의 물리 데미지를 줍니다.

스킬 사용 :

- '공격 스킬' 또는 '방어 스킬'을 선택하여 사용합니다.

- 스킬 사용 시 정해진 MP가 소모됩니다. (MP 부족 시 사용 불가)

스킬의 효과(데미지, 방어량, 버프/디버프)는 캐릭터의 attackDamage 또는 magicForce 스탯에 기반하여 계산됩니다.

### 3. 성장 및 스탯 분배
레벨업 스탯 :

- 레벨업시 자동으로 공격력과, 체력스탯이 오른다

### 4. 클래스(직업) 및 전직 시스템
전직 트리 :

- 모든 플레이어는 검사 로 시작합니다.

- 2차 (Lvl 10): 기사 (Knight)

- 3차 (Lvl 20): 대검사 (Greatsword Master) 또는 성기사 (Paladin)

- 4차 (Lvl 30): 용기사 (Dragon Knight) (대검사 상위) 또는 성검사 (Holy Knight) (성기사 상위)

전직 조건:

- '레벨업 기능 목록'에 정의된 대로, 10, 20, 30레벨에 도달 시 전직 퀘스트가 발생하며, 완료 시 전직합니다.

클래스별 스킬 보유 :

- 검사 (1차): 공격 스킬 1, 방어 스킬 1

- 기사 (2차): 공격 스킬 1, 방어 스킬 1, 

- 대검사/성기사 (3차): 공격 스킬 2 (+1), 방어 스킬 2 (+1), 

- 용기사/성검사 (4차): 공격 스킬 2, 방어 스킬 2, 궁극기 스킬 1

### 5. 스킬 유형 정의
공격 스킬 :

- MP를 소모하여 적에게 attackDamage 또는 abilityPower 기반의 강력한 데미지를 입힙니다.

- (예: 강력 베기 - 물리, 신성 타격 - 마법)

방어 스킬 :

- MP를 소모하여 자신을 보호하거나 강화합니다.

- (예: 철벽 방어 - 1턴간 방어력 2배, 치유의 빛 - HP 회복)

궁극기 스킬 :

- MP를 대량 소모하며, 전투의 흐름을 바꿀 수 있는 매우 강력한 스킬입니다.

- (예: 드래곤 브레스, 최후의 심판)



### 6. 장비 시스템
장착 부위 :

- 무기 : (예: 검) attackDamage와 magicForce를 크게 증가시킴

- 방어구 : (예: 갑옷, 투구, 신발) 방어력과 최대 HP를 증가시킴

- 장신구 : (예: 반지, 목걸이) 특수 능력(예: MP 회복률 증가)이나 스탯을 소폭 증가시킴
