## java - RpgGame

## 턴제 RPG텍스트 게임

### 게임 개요 
- 용사는 각 스테이지의 몬스터를 무찌르고 월드당 1개의 황금 무기를 얻어서  마지막 스테이지 최종보스를 무찌르고 세상을 구하는 게임


### 게임 (전직)직업
검사 -> 기사 -> 성기사 -> 성전사 -> 대검사 -> 용기사

### 기본스탯
체력 = 50

마나 = 10

공격력 = 10

마력 = 10

---
### 캐릭터 정보
#### 검사 (기본)
- 밸런스형 초반 근접 전사, 초반 밸런스형. 단순 근접 공격, 방어 스킬 1개 보유
기사 : 방어·생존력 강화형, 방어 강화, 도발·회복 스킬 보유 (턴제에서 ‘턴 버는’ 역할)

#### 성기사 (1차 전직)
- 물·마 복합형. 회복과 신성 공격으로 안정적인 전투 지속
#### 성검사 (2차 전직)
- 반사 피해 + 높은 내구도, 장기전 특화. 버프 보유, 마법 위주의 전투 가능

#### 대검사 (1차 전직)
- 순수 물리딜 극대화형, 순수 물리딜러. 한 방이 강하지만 방어와 MP는 낮음
#### 용기사 (2차 전직)
- 폭발적 공격력 + 기동성 강화형, 필살기 중심. HP와 공격력 높고, 턴마다 ‘용의 분노’ 스택형 전투

---

### 기본 게임구조

- 나 공격한번(1턴소모) 상대 공격한번(상대도 1턴소모)

- 총 5월드  월드당 10스테이지 1~4스테이지 일반몬스터 한 마리 5, 10스테이지는 중간보스 몬스터, 보스몬스터

- 보스 몬스터를 무찌르면 황금 무기조각 하나획득

레벨형  1->10->20->30->40
- 1월드 클리어  1차전직 10
- 2월드 클리어  2차전직 20
- 3월드 클리어  3차전직 30
- 4월드 클리어  4차전직 40
- 5월드

---

레벨업할 때마다  능력치 증가

1포인트당 10씩 증가

10포인트부여 (체력, 방어력, 공격력, 마력 부여가능)

전직시 직업 특색에 맞게 능력치 차등부여

- 검사 -> 기사 : 공격력, 방어력, 체력
- 기사 -> 성기사 : 마력, 방어력, 체력
- 기사 -> 대검사 : 공격력, 체력, 방어력
- 성기사 -> 성검사 : 마력 방어력 체력
- 대검사 -> 용기사 : 공격력, 체력, 방어력

---

### 몬스터 정보

그림은 아스키 아트로

- 기본 몬스터 1~4스테이지, 6~9스테이지 등장
- 중간 보스 몬스터 : 5스테이지 등장
- 보스 몬스터 : 10스테이지 등장
- 최종 보스 몬스터 : 5월드 11스테이지 최종보스 등장


### 상태이상 요소
n%확률로 상태이상이 걸릴 수 있다.
- 기절
- 감전

### 아이템 / 장비 시스템
- 무기
- 방어구
- 반지

### 스킬 목록

- 검사 : 공격스킬, 방어스킬 1개씩
- 기사 : 공격스킬 1개, 방어스킬 1개, +패시브 스킬 1개
- 성기사(대검사) : 공격스킬 1개 방어스킬 1개 패시브 스킬 1개 +궁극기 1개
- 성검사(용기사) : 공격스킬 1개 방어스킬 1개 패시브 스킬 1개 +궁극기(강화) 1개
### 참고요소
